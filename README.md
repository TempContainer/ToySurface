# ToySurface

A toy water surface simulation use naive sin wave and Gerstner wave.

naive sin wave:

![naive](image.png)

Gerstner Wave (ugly texture mapping make it like rag...):

![Gerstner](image-1.png)

## Usage

- `Left Mouse` to change viewpoint.
- `Right Mouse` to change scale.

## Dependencies

- gcc (C++20 required)
- glut
- glm
